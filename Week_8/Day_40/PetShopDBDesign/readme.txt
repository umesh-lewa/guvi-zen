Collections Relationship

One Customer -> Can have multiple adoptions
Relationship = One to Many
Customer Document will have reference to Multiple adoption ObjectIDs

One Adptoption -> Can contain multiple pets
Relationship = One to Many
Adptoption Document will have reference to Multiple pet ObjectIDs

One Employee -> Can care of multiple pets
Relationship = One to Many
Employe Document will have reference to Multiple pet ObjectIDs

One Employees -> Can carry out multiple adoptions
Relationship = One to Many
Employe Document will have reference to Multiple adoption ObjectIDs
